# HoloMotion.HTML

Hand-controlled 3D particle system using Three.js and TensorFlow.js.

## Features
- ✋ **Open/Close Hand**: Expand/Shrink particles
- 🤏 **Pinch (Thumb+Index)**: Switch shape template
- ↔️ **Move Left/Right**: Change colors
- 🔄 **Move Hand**: Rotate 3D system

## Setup
Just open `index.html` in a modern browser with webcam access.
